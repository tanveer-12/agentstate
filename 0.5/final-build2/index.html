    Product vision

    agentstatelib is a stateful coordination layer for multi-agent AI systems. It gives multiple agents a shared, typed world model and handles every concern between the agent functions and the application layer:

    • Shared typed state — SharedState (Pydantic v2) is the single source of truth. Agents never mutate it directly.
    • StatePatch pipeline — agents return structured patches; the library validates, conflict-resolves, and applies them atomically.
    • Event-sourced audit log — every patch, conflict, checkpoint, and error is recorded as a typed immutable event. Any past state is reproducible from the log.
    • AgentGraph — round-based parallel execution engine with conflict detection, invariant checking, and approval gating.
    • Context slicing — agents receive only the state paths they declared, limiting context window usage and enforcing read boundaries.
    • Pluggable persistence — InMemoryStore (tests), SQLiteStore (single process), PostgreSQLStore (production).
    • Checkpointing — snapshot save/load with on-disk persistence for fault recovery.
    • OpenTelemetry — optional distributed tracing with no-op fallback; zero import overhead if otel is not installed.
    • Rich terminal dashboard — live workflow monitor with optional rich dependency.
    • FastAPI web dashboard — HTML/JS single-page app served from the same process, with SSE streaming, approval modals, and a turn-detail trace view.
    • Local model reliability — LLMAgent base class with retry-with-correction loop, validation, and per-attempt event emission.
    • Human-in-the-loop — approval gates on edges; HumanApprovalRequested/Resolved events; REST API and programmatic resolution via resume_from_approval.

    Competitive positioning: typed state + conflict detection + event sourcing + replay debugger is a combination that does not exist in LangGraph, CrewAI, or AutoGen.

    state.py — IMPLEMENTED

    • WorkflowStatus — Literal["running", "complete", "failed", "paused"]
    • Task — id, goal_id, description, status, result, created_at, updated_at
    • Goal — id, description, status, created_at
    • Artifact — id, produced_by, artifact_type, content, created_at

    All Pydantic v2, full docstrings, all exported.

    patch.py — IMPLEMENTED

    • StatePatch — patch_id, agent_id, target, value, reason, timestamp, priority
    • set_nested(obj, path, value) — creates intermediate dicts automatically
    • get_nested(obj, path) — returns None on missing, never raises
    • apply_patch(state, patch) → new SharedState — original never mutated

    events.py — IMPLEMENTED (16 event types)

    All inherit BaseStateEvent which carries event_id, workflow_id, agent_id, timestamp, schema_version.

    Discriminated union via type field. event_adapter = TypeAdapter(StateEvent) for JSON deserialization.

    Event Phase Purpose
    WorkflowStarted 1 Graph.run() begins
    WorkflowCompleted 1 Graph.run() finishes
    PatchApplied 1 Winning patch applied
    ConflictDetected 1 Two patches collide in same round
    CheckpointSaved 1 save_checkpoint() completes
    AgentErrored 1 Agent raised unhandled exception
    ContextSliced 2A Agent context prepared, paths + size logged
    PromptAssembled 2A Full prompt text recorded before model call
    ModelCalled 2A Pre-call marker, links to ModelReturned via call_id
    ModelReturned 2A Raw response, latency, tokens, estimated cost
    ValidationFailed 2A JSON/schema parse error, includes raw_output
    RetryAttempted 2A Correction attempt start, previous_error attached
    ToolCalled 2A External tool invoked, tool_input logged
    ToolReturned 2A Tool result, latency, success flag
    HumanApprovalRequested 2E Graph paused, patch embedded for review
    HumanApprovalResolved 2E Human decision recorded (approved/rejected/modified)

    agentstatelib/coordination/

    conflicts.py — IMPLEMENTED

    • ConflictRecord — conflict_id, path, winner_agent_id, loser_agent_id, resolution_strategy, existing_patch, incoming_patch
    • BatchResolutionResult — winners: list[StatePatch], conflicts: list[ConflictRecord]
    • ConflictResolver — Protocol: resolve(existing, incoming) → StatePatch
    • LastWriteWins — higher timestamp wins (default)
    • PriorityBased — higher StatePatch.priority wins
    • RejectIncoming — first writer wins, incoming always loses
    • ConflictDetector — resolve_batch(patches) for atomic round resolution; submit(patch) for sequential use; reset() clears history

    invariants.py — IMPLEMENTED

    • InvariantViolation — rule_name, description, severity: Literal["warning", "error"]
    • InvariantChecker — Protocol: check(state) → list[InvariantViolation]
    • TasksReferenceExistingGoals — tasks with goal_id must reference a real goal
    • CompletedGoalsHaveNoBlockingTasks — complete goals must not have pending tasks
    • check_all(state, checkers) — runs all checkers, error violations halt graph with RuntimeError

    agentstatelib/router/

    types.py — IMPLEMENTED

    • AgentFn = Callable[[dict[str, Any]], Awaitable[StatePatch | list[StatePatch]]]
    • EdgeCondition = Callable[[dict[str, object]], bool]

    context.py — IMPLEMENTED

    • slice_state(state, include_paths) → dict[str, Any]
    • Always includes workflow_id, workflow_type, goal in returned dict
    • Handles nested dotted paths ("facts.key", "tasks.t1.status")
    • Empty include_paths returns full state dict

    graph.py — IMPLEMENTED

    Key types: - _Node(frozen dataclass) — agent_id, fn, context_keys - _Edge(frozen dataclass) — from_agent, to_agent, condition, approval_required - PendingApproval(frozen dataclass) — workflow_id, state, patch, agent_id, description - WorkflowEvent — union type alias for queue-able events - EventQueue = asyncio.Queue[WorkflowEvent]

    AgentGraph methods: - __init__(store, max_concurrent, conflict_resolver, invariant_checkers) — sets up semaphore, ConflictDetector, pending_approvals: dict[str, PendingApproval] - node(agent_id, context) — decorator, registers _Node - edge(from_agent, to_agent, condition, approval_required) — creates _Edge with optional approval predicate - add_invariant(checker) — appends to invariant checker list - run(state, start, event_queue) → SharedState — full round loop with OTel span - resume_from_approval(approval_id, decision, modified_patch) → SharedState | None — pops from pending_approvals, emits HumanApprovalResolved, applies patch if approved/modified - _execute_round(...) — parallel agent execution, conflict resolution, approval gating, invariant checks - _run_agents_parallel(...) — asyncio.gather with semaphore, emits ContextSliced per agent - _next_round(just_ran, state_dict) — edge condition evaluation, deduplication - _emit(event, queue) — store.append + queue.put_nowait - _build_conflict_event(workflow_id, record) — ConflictRecord → ConflictDetected

    agentstatelib/memory/

    store.py — IMPLEMENTED

    StateStore protocol (5 methods): append, get_workflow, since, count, list_workflows

    InMemoryStore: - dict[str, list[StateEvent]] backing store - All 5 protocol methods implemented - list_workflows returns keys in reverse insertion order

    SQLiteStore: - aiosqlite async backend - Table: events(id, event_id, workflow_id, type, schema_version, data, timestamp) - Index on workflow_id - Auto-migration: adds schema_version column to existing databases - list_workflows orders by MAX(id) DESC

    PostgreSQLStore: - Optional asyncpg backend (pip install asyncpg) - Pool created lazily on first use - Mutual-recursion-free _init_db(pool) pattern - close() for graceful pool shutdown - HAS_ASYNCPG flag prevents import errors when asyncpg is absent

    • load_latest_checkpoint(workflow_id, directory) → Checkpoint | None
    • load_checkpoint(checkpoint_id, directory) → Checkpoint

    replay.py — IMPLEMENTED

    Running the project

    # Install all dev dependencies
pip install -e ".[api,otel,dashboard,contrib,dev]"

# Type check (must be 0 errors)
python -m mypy agentstatelib --ignore-missing-imports --no-error-summary

# Run all tests
python -m pytest tests/ -q --tb=short

# Start the API server
$env:AGENTSTATE_API_KEYS = "dev-key-123"
uvicorn agentstatelib.api.app:app --reload --port 8000

# Open the web dashboard
# http://localhost:8000/dashboard
